# AetherControl Core

Control plane API for projects, agents, tasks, approvals, and policy checks.

## Planned modules

- `projects`
- `agents`
- `tasks`
- `approvals`
- `policies`
- `integrations`
- `audit`

## Planned endpoints (v1)

- `POST /projects`
- `GET /projects/:projectId/overview`
- `POST /agents`
- `POST /tasks`
- `POST /tasks/:taskId/approve`
- `POST /tasks/:taskId/reject`
- `POST /budgets/check`
